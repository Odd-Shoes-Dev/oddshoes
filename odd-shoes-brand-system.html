08 · Usage Guidelines
Do's and Don'ts
✓ Do
Keep the mismatch

"Odd" must always be in Playfair Display italic. "shoes" must always be in JetBrains Mono regular. This contrast IS the logo. Never set both in the same font.

✗ Don't
Match the fonts

Never set "Odd shoes" in a single typeface. The whole brand is built on the deliberate mismatch. If both words look the same, the meaning is lost.

✓ Do
Use the cursor

The blinking coral cursor is a signature element. Include it on digital/screen applications where animation is possible. On print, use a static coral line.

✗ Don't
Recolour Solé

Solé's body is always dark (#1A1A1A) or white on dark backgrounds. The beak is always coral. The shoes are always mismatched (coral sneaker + black dress shoe).

✓ Do
Clear space

Maintain minimum clear space equal to the height of the "O" on all sides of the logo. More space is always better. Let the brand breathe.

✗ Don't
Add effects

No drop shadows, gradients, bevels, outlines, or 3D effects on the wordmark. The logo's power is in its clean typographic contrast. Keep it flat and sharp.
